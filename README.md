# ChatLuna Standalone

独立版 ChatLuna 后端，基于 Cordis 构建。

## 项目结构

- [@chatluna/standalone](./packages/webui/README.MD)

  ChatLuna 独立版主插件。

- [@chatluna/server](./packages/server/README.MD)

  ChatLuna server 实现。提供后端 API 给 Web UI 和其他应用的后端。
