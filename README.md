# ChatLuna Standalone

独立版 ChatLuna，并且可使用 Web UI 访问。

基于 Cordis 构建。

## 项目结构

- [@chatluna/web-ui](./packages/webui/README.MD)

  ChatLuna WebUI 实现。

- [@chatluna/server](./packages/server/README.MD)

  ChatLuna server 实现。提供给 web-ui 和其他应用的后端。
